# Project 1: The Digital Architect Hub (Frontend Interface)

## 📌 Overview
Welcome to Project 1 of the DecodeLabs Full Stack Development Industrial Training. This project represents the "Skin" of our application—a pure, framework-free, responsive frontend interface built with absolute architectural integrity.

The core objective was to master frontend adaptability, strict micro/macro layout architectures, and universal accessibility without relying on external libraries like React or Bootstrap.

## ✨ Key Features
* Mobile-First Paradigm: Fluid configurations scaling seamlessly across mobile (360px), tablet (768px), and desktop (1024px+) viewports.
* Semantic Integrity: Utilization of proper HTML5 landmarks (`<header>`, `<nav>`, `<main>`, `<article>`, `<aside>`) for superior accessibility and AI parsing.
* 2025 Aesthetics Blueprint: Implemented a grounded visual theme utilizing the strict DecodeLabs palette:
    * `#A5856F` (Mocha Mousse - Stability)
    * `#A0D4E0` (Ethereal Blue - Trust)
    * `#F2F0EA` (Moonlit Grey - Refinement)
* **Advanced Layout Orchestration:** Macro layouts powered by CSS Grid and micro-components managed by CSS Flexbox.
* **Vanilla JS Logic:** Client-side state management for mobile navigation toggles and dynamic theme shifting (Glassmorphism UI).

## 🛠️ Technology Stack
* HTML5 (Structural Semantics)
* CSS3 (Grid, Flexbox, Custom Variables, Fluid Typography)
* Vanilla JavaScript (ES6+ Object-Oriented DOM Manipulation)

## 🚀 Getting Started
Since this project adheres strictly to the "0% External Dependencies" mandate, no build steps are required.

1.  Clone the repository.
2.  Open the project directory.
3.  Launch `index.html` in any modern web browser.
     *Tip: Use the VS Code "Live Server" extension for the best development experience.
